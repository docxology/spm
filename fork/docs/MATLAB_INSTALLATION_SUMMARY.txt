Step 1: Install MATLAB
  → Visit: https://www.mathworks.com/downloads/
  → Download for macOS
  → Run installer
  → Install to /Applications/

Step 2: Verify Installation
  → Open Terminal
  → Run: /Applications/MATLAB_R2024a.app/bin/matlab -version
  → Should display MATLAB version

Step 3: Add to PATH (Optional)
  → Edit: nano ~/.zshrc
  → Add: export PATH=/Applications/MATLAB_R2024a.app/bin:$PATH
  → Save & reload: source ~/.zshrc

Step 4: Setup SPM25
  → Terminal: cd /Users/4d/Documents/GitHub/spm
  → Launch MATLAB: matlab
  → Run: spm_setup()
  → Should see: "SPM setup complete!"

Step 5: Verify SPM25
  → Run: spm_dev_info()
  → Or: spm_run_tests('core')
  → Or: spm fmri (launch GUI)
